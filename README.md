## 为什么要用VPS建站，而不是直接买虚拟主机？

这是新手最常问的问题之一。

虚拟主机（Shared Hosting）就像合租公寓——你和一堆陌生人挤在同一台服务器上，共享CPU、内存、带宽。价格便宜，但上限也低：邻居一跑高流量，你的网站也跟着变慢；想装个特定版本的PHP，对不起，管理员说了算。

VPS（Virtual Private Server）则是自己租了一套独立公寓。CPU、内存、存储都是你的，root权限随便用，想装什么软件装什么软件，性能稳定，不受其他用户影响。

近几年VPS价格下降幅度挺大的，最便宜的方案每月不到三美元就能拿到，性价比已经完全碾压传统虚拟主机。对于想认真做网站的人来说，VPS几乎是唯一值得考虑的选项。

## VPS建站需要准备什么？

在动手之前，先把要用到的东西列清楚：

1. **一台VPS**：这是整个建站的基础，下文会重点讲如何选择
2. **一个域名**：比如 `yoursite.com`，在Namecheap、GoDaddy等平台购买，每年十几到几十块人民币不等
3. **SSH客户端**：用来连接服务器，Windows推荐用 Termius 或 PuTTY，Mac/Linux直接用系统终端
4. **一点点耐心**：整个流程首次操作大概需要1-2小时

## 第三步：连接服务器

拿到IP地址和root密码之后，打开SSH客户端连接：

bash
ssh root@你的VPS_IP地址


第一次连接会提示确认主机指纹，输入 `yes` 回车，然后输入密码就进去了。

进来之后第一件事——**改密码，顺便更新系统**：

bash
passwd
apt update && apt upgrade -y


这两步不做容易被扫描器暴力破解，真的每天都有机器人在扫。

## 第五步：安装WordPress

如果选了手动环境（宝塔面板路线），装完LNMP之后在面板里操作：

1. 点左侧「网站」→「添加站点」
2. 域名填你的域名（比如 `example.com` 和 `www.example.com` 都填上）
3. 数据库类型选MySQL，设置数据库名和密码（记下来，后面用）
4. PHP版本选8.1或8.2
5. 点「提交」，站点创建完成

然后去WordPress官网下载安装包，上传到站点目录（宝塔的文件管理器就能传），访问你的域名就会自动弹出WordPress安装向导，按提示填数据库信息，三分钟内完成安装。

## 第六步：绑定域名

网站装好了，还需要把域名指向你的服务器IP。

进入你的域名注册商后台，找到DNS管理，添加一条A记录：

- **主机名**：`@`（代表根域名）
- **指向**：你的VPS IP地址
- **TTL**：默认就行

同样的方式再加一条 `www` 的A记录，指向同一个IP。

DNS生效通常需要几分钟到几小时不等。生效之后在浏览器输入你的域名，就能看到WordPress的安装向导或者已经跑起来的网站了。

## 建站之后还要做什么？

装完WordPress、绑完域名、搞好HTTPS，这只是网站搭起来了，离"好网站"还有一段路：

- **选一个好主题**：直接影响网站外观和速度，免费的有Astra、GeneratePress，付费的有Kadence、Blocksy
- **安装必备插件**：缓存插件（LiteSpeed Cache或WP Super Cache）、SEO插件（Yoast或RankMath）、安全插件（Wordfence）
- **提交到Google Search Console**：让搜索引擎更快收录你的网站
- **定期备份**：虽然Evoxt自带每周备份，但关键内容建议额外备份到本地或云存储
